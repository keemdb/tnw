TOPIC 등록
모니터 할 토픽 문자열을 지정. 문자열은 $ SYS로 시작할 수 없으며 multi-level 와일드 카드
(예 : / animals / dogs / #)로 끝나야 한다.
모든 토픽에 대한 통계를 집계하려면 single multi-level 와일드 카드 (#)를 지정, 주제에 하위 토픽이
없는 경우 모니터는 해당 주제에 대한 데이터만 표시한다.
예를 들어, labradors에 하위 토픽이 없는 경우 / animals / dogs / labradors / #는 / animals
/ dogs / labradors 토픽만 모니터링 한다.
