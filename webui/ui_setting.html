WEB UI SETTINGS

Web UI에 대한 설정을 구성
Publish Policies | 웹 UI 통신을 위한 IP 주소 및 포트를 선택해라. 두 값 중 하나를 변경하면 새 IP 주소와 포트에 로그인해야 한다.
  • IP Address
  • Port * 유효한 포트는 1 ~ 8999, 9087 또는 9100 ~ 65535이다.
Session Timeout and Refresh Time | 브라우저와 서버 간의 활동을 기반으로 사용자 로그인 세션을 제어 및 그래프 리플래쉬 시간을 설정 한다.
  • Session Timeout miniutes
  • Refresh Time seconds
HTTP Read Timeout | 첫 번째 읽기가 발생한 후 소켓에서 읽기 요청이 완료 될 때까지 기다리는 최대 시간이다.
  • Session Timeout miniutes * 1 ~ 300초(5분) 사이의 값이다.
Service | 웹 UI에 대한 지원 정보를 생성하도록 추적 레벨을 설정
  • Enable trace
  • Trace Specification
